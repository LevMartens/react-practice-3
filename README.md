## About The Project

### Straight Line 

Straight Line is a game where the objective is to walk a straight line through a rough terrain. 
The gps will track how much you deviate from that line and will give a score in the end. 

### Build With

* React Native
* Expo
* Graphql
* AWS cloud services
* Redux
* Material UI
* react-native-maps
* react-native-paper
* lodash
